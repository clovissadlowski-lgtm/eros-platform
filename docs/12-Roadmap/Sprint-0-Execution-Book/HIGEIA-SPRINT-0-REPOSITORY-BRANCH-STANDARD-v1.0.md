# Higeia — Sprint 0 Repository and Branch Standard

## Recommended code structure

```text
src/
tests/
scripts/
config/
infrastructure/
docs/
```

## Branch model

- main
- development or trunk-based alternative
- feature/*
- fix/*
- chore/*

## Pull request requirements

- clear title;
- linked story;
- acceptance criteria;
- tests;
- security/privacy impact;
- evidence;
- reviewer.

## Protected assets

- secrets;
- production configuration;
- private keys;
- raw health data;
- provider credentials.
