# Higeia — Governance Quick Start

## I have an idea

Use:

1. Use Case Register
2. Use Case Brief
3. Value Hypothesis
4. Prioritization Matrix

## I want to change a model

Use:

1. Model Change Record
2. Model Card
3. Evaluation Framework
4. Release Gate
5. Deployment Record

## I want to change a prompt

Use:

1. Prompt Registry
2. Prompt Change Record
3. Regression Evaluation
4. Approval Record

## A control failed

Use:

1. Finding Record
2. Escalation Matrix
3. Exception Record if temporary
4. Remediation Plan
5. Evidence Register

## An incident occurred

Use:

1. Incident Record
2. Immediate containment
3. Escalation
4. Investigation
5. Corrective action
6. Closure review

## I want to release Alpha

Use:

1. Readiness Matrix
2. Blocker Register
3. Verification Plan
4. Management Review
5. Alpha Decision Record
