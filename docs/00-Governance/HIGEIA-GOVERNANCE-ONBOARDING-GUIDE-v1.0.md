# Higeia — Governance Onboarding Guide

## Required reading

- Governance Manual
- Constitution and Principles
- Product scope
- Relevant domain framework
- Incident and escalation rules

## Required actions

- Confirm role.
- Confirm owner and backup owner.
- Complete required training.
- Receive access.
- Review applicable controls.
- Review current risks.
- Review active exceptions.
- Sign or confirm authorization when required.

## First-week checklist

- [ ] Read core documents.
- [ ] Understand repository structure.
- [ ] Meet domain owner.
- [ ] Review current roadmap.
- [ ] Review active blockers.
- [ ] Review escalation path.
- [ ] Confirm tools and access.
