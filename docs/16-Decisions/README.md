## Regra

Toda mudança relevante de produto, negócio, IA, dados, segurança ou arquitetura deve ser registrada.

Uma decisão deve conter:

- contexto;
- decisão;
- justificativa;
- consequências;
- riscos;
- alternativas;
- condição de revisão.
