# Higeia — Core User Journeys

## Nutritionist

1. Create account.
2. Confirm professional profile.
3. Invite patient.
4. Monitor onboarding.
5. Review anamnesis.
6. Receive weekly check-in.
7. Review AI summary.
8. Correct or approve.
9. Use in consultation.
10. Review longitudinal progress.

## Patient

1. Receive invitation.
2. Create access.
3. Review consent and privacy.
4. Complete anamnesis.
5. Submit weekly check-in.
6. Add measurements.
7. Add text report.
8. Upload allowed photos.
9. Correct data.
10. Manage privacy and support requests.

## Support/Operations

1. Monitor alerts.
2. Review incidents.
3. Apply escalation.
4. Restrict feature if needed.
5. Register decision.
6. Close with evidence.
