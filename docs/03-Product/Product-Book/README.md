## Uso

O Product Book define visão, problema, usuários, MVP e Alpha.

O Alpha Scope delimita o experimento.

O Feature Inventory controla funcionalidades.

O Use Case Catalog detalha casos prioritários.

As User Journeys mostram fluxos.

As Metrics definem sucesso.

O Decision Register registra decisões.

O Requirements Baseline orienta arquitetura e engenharia.

## Regra

Nenhuma funcionalidade deve entrar no Alpha sem estar classificada como incluída, restrita, manual, excluída ou adiada.
