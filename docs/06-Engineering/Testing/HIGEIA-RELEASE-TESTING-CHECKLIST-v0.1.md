# Higeia — Release Testing Checklist

**Versão:** 0.1  

## Antes da release

- [ ] Escopo congelado.
- [ ] PRs aprovadas.
- [ ] Lint aprovado.
- [ ] Typecheck aprovado.
- [ ] Testes unitários aprovados.
- [ ] Testes de integração aprovados.
- [ ] E2E crítico aprovado.
- [ ] Regressão aprovada.
- [ ] Segurança revisada.
- [ ] Privacidade revisada.
- [ ] IA avaliada.
- [ ] Migrations revisadas.
- [ ] Backup validado.
- [ ] Rollback disponível.
- [ ] Staging validado.
- [ ] Release notes prontas.
- [ ] Monitoramento ativo.
- [ ] Responsável pela release definido.

## Depois da release

- [ ] Smoke test executado.
- [ ] Logs revisados.
- [ ] Métricas revisadas.
- [ ] Alertas revisados.
- [ ] Erros críticos ausentes.
- [ ] Fluxos principais confirmados.
- [ ] Período de observação concluído.
