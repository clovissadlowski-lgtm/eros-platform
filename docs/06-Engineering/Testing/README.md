# Testing

Esta pasta concentra a estratégia e os instrumentos de teste do Higeia.

## Documentos

- `HIGEIA-TESTING-STRATEGY-v0.1.md`
- `HIGEIA-CRITICAL-TEST-MATRIX-v0.1.md`
- `HIGEIA-BUG-REPORT-TEMPLATE-v0.1.md`
- `HIGEIA-RELEASE-TESTING-CHECKLIST-v0.1.md`

## Uso

A estratégia define os padrões gerais.

A matriz relaciona riscos críticos aos testes necessários.

O template de bug padroniza o registro de defeitos.

O checklist de release deve ser usado antes e depois de cada publicação.

## Regra

Nenhum gate crítico pode ser ignorado apenas para acelerar uma entrega.
