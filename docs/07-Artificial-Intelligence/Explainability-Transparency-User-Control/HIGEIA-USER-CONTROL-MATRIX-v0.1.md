# Higeia — User Control Matrix

**Versão:** 0.1

| Object | View | Correct | Dispute | Delete | Disable | Human Review | Export |
|---|---|---|---|---|---|---|---|
| Profile data | Yes | Yes | Yes | Conditional | No | Conditional | Yes |
| Memory | Yes | Yes | Yes | Yes | Yes | Conditional | Yes |
| AI inference | Yes | Contextual | Yes | Conditional | No | Yes | Yes |
| AI hypothesis | Yes | Contextual | Yes | Conditional | No | Yes | Yes |
| Recommendation | Yes | No | Yes | No | No | Conditional | Yes |
| Score | Yes | Source data | Yes | Conditional | No | Conditional | Yes |
| Uploaded document | Yes | Replace | Yes | Yes | No | No | Yes |

## Regras

- “Conditional” depende de obrigação, retenção e impacto.
- Exclusão deve considerar derivados.
- Correção de dado-fonte deve reprocessar resultados relevantes.
