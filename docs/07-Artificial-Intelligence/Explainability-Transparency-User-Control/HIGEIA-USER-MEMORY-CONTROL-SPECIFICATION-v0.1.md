# Higeia — User Memory Control Specification

## Purpose

Definir o comportamento do painel de memórias do usuário.

## Memory card

Cada memória deve mostrar:

- categoria;
- conteúdo;
- origem;
- data;
- finalidade;
- confiança;
- expiração;
- status.

## Actions

- View details
- Edit
- Confirm
- Dispute
- Disable
- Delete
- Prevent similar memory

## Rules

- Ações devem ser simples.
- Exclusão deve ser confirmada.
- Correção deve propagar.
- Dados sensíveis devem ter proteção adicional.
- Memória inferida deve estar claramente identificada.
- Rejeição não deve prejudicar o acesso principal ao produto.
