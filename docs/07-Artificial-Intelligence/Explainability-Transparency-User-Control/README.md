## Uso

O framework define as regras gerais.

O explanation template estrutura explicações.

A memory specification define controles de memória.

O dispute record registra contestações.

O human review request formaliza revisão humana.

O checklist controla transparência antes do release.

O labeling standard padroniza rótulos.

A control matrix define ações disponíveis.

## Regra

Toda saída relevante de IA deve deixar claro o que é fato, cálculo, inferência ou hipótese, quais limitações existem e como o usuário pode corrigir, contestar ou pedir revisão.
