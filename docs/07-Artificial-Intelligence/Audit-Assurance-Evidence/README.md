## Uso

O framework define as regras gerais.

O control register inventaria controles.

O evidence register inventaria evidências.

O audit plan organiza auditorias.

O control test record documenta testes.

O finding record registra falhas.

O action plan controla remediação.

O evidence pack index prepara due diligence.

O self-assessment mede maturidade.

O audit report comunica conclusões.

## Regra

Nenhum controle crítico deve ser considerado efetivo sem owner, evidência, teste, resultado e tratamento de falhas.
