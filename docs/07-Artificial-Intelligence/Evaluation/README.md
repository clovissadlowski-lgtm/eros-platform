# AI Evaluation

Esta pasta concentra a estrutura de avaliação de inteligência artificial do Higeia.

## Documentos

- `HIGEIA-AI-EVALUATION-FRAMEWORK-v0.1.md`
- `HIGEIA-AI-EVALUATION-RUBRIC-v0.1.md`

## Uso

O framework define os princípios, métricas e gates.

A rubrica orienta a revisão de cada resposta.

O dataset template organiza os casos de teste.

O evaluation run registra uma execução comparativa.

O release gate checklist decide se a versão pode avançar.

## Regra

Nenhuma versão de IA de alto risco deve chegar a usuários sem dataset, avaliação, revisão humana, safety, privacidade e rollback.
