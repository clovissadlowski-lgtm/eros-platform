## Uso

O framework define as regras gerais.

A RACI distribui responsabilidades.

O decision record documenta decisões.

A authority matrix define alçadas.

A delegation record formaliza delegação.

O conflict record documenta conflitos.

O exception record controla exceções.

O committee charter define o comitê.

O meeting record registra reuniões.

## Regra

Toda decisão relevante deve ter accountable, evidência, alçada, validade, possibilidade de bloqueio e trilha de auditoria.
