## Uso

O framework define o modelo integrado.

O Master Register conecta objetos.

O Control Catalog consolida controles.

O Evidence Register consolida evidências.

O Identifier Standard padroniza IDs.

O Calendar define cadência.

O Gate Register consolida aprovações.

O Meeting Record registra decisões.

O Exception Record controla desvios.

A Escalation Matrix define resposta.

O Dashboard consolida indicadores.

A Maturity Assessment mede evolução.

A Management Review orienta decisões executivas.

## Regra

A governança deve funcionar como sistema operacional, não como biblioteca passiva de documentos. Todo objeto crítico deve possuir ID, owner, status, fonte, evidência e próxima revisão.
