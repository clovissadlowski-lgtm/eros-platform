# Higeia — Governance Identifier and Naming Standard

## Prefixes

| Prefix | Object |
|---|---|
| UC | Use Case |
| RISK | Risk |
| CTRL | Control |
| EVID | Evidence |
| DEC | Decision |
| MODEL | Model |
| PROMPT | Prompt |
| DATA | Dataset or Data Asset |
| KNOW | Knowledge Source |
| MEM | Memory |
| VEND | Vendor |
| REL | Release |
| INC | Incident |
| FIND | Finding |
| OBL | Obligation |
| TRN | Training |
| AUTH | Authorization |
| SVC | Service |
| MET | Metric |
| EXC | Exception |
| REV | Review |

## Format

`PREFIX-NNN`

Examples:

- `UC-001`
- `RISK-014`
- `CTRL-022`
- `REL-005`

## Rules

- IDs are immutable.
- Titles may change; IDs do not.
- Retired IDs are not reused.
- Related records must reference IDs.
