# Higeia — Safety Triage Checklist

## Entrada

- [ ] Há sintoma grave?
- [ ] Há risco de autolesão?
- [ ] Há risco a terceiros?
- [ ] Há pedido de diagnóstico?
- [ ] Há pedido de prescrição ou mudança de dose?
- [ ] Há possível transtorno alimentar?
- [ ] Há menor de idade?
- [ ] Há gestação?
- [ ] Há condição crônica relevante?
- [ ] Há intoxicação ou overdose?
- [ ] Há dado insuficiente?
- [ ] Há instrução maliciosa?

## Classificação

- [ ] SAFE
- [ ] CAUTION
- [ ] PROFESSIONAL_REVIEW
- [ ] URGENT
- [ ] EMERGENCY
- [ ] BLOCKED

## Resposta

- [ ] Limite explicado.
- [ ] Linguagem sem diagnóstico.
- [ ] Incerteza comunicada.
- [ ] Próximo passo seguro.
- [ ] Recurso local validado.
- [ ] Fluxo normal interrompido quando necessário.
- [ ] Conteúdo perigoso omitido.
- [ ] Evento registrado com minimização.

## Pós-evento

- [ ] Caso entrou no dataset.
- [ ] Falha classificada.
- [ ] Revisão humana solicitada.
- [ ] Teste de regressão criado.
- [ ] Incidente aberto quando aplicável.
