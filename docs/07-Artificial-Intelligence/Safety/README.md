# AI Safety

Esta pasta concentra os limites clínicos e os controles de segurança da IA do Higeia.

## Documentos

- `HIGEIA-AI-SAFETY-CLINICAL-BOUNDARIES-v0.1.md`
- `HIGEIA-CLINICAL-BOUNDARY-MATRIX-v0.1.md`
- `HIGEIA-SAFETY-RESPONSE-PATTERNS-v0.1.md`
- `HIGEIA-SAFETY-TRIAGE-CHECKLIST-v0.1.md`
- `HIGEIA-CLINICAL-SAFETY-REVIEW-RECORD-v0.1.md`

## Uso

O documento principal estabelece os limites.

A matriz resume o que é permitido ou proibido.

Os padrões de resposta são rascunhos para validação.

O checklist orienta classificação e escalonamento.

O registro documenta aprovação clínica e de safety.

## Regra

Nenhum fluxo clínico, de crise, medicação ou alto risco deve chegar a usuários sem validação profissional, testes críticos, fallback, monitoramento e revisão regulatória.
