## Uso

O framework define a política geral.

O logging standard define eventos e campos.

O metrics catalog organiza indicadores.

A dashboard specification define painéis.

O alert catalog organiza alertas.

A política de custo define orçamento e limites.

O relatório semanal acompanha operação.

## Regra

Nenhum fluxo de IA deve ir para produção sem rastreabilidade de versão, custo, latência, falhas, safety, fallback e owner.
