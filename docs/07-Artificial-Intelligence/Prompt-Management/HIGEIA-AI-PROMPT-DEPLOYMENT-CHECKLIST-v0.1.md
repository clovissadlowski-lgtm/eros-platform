# Higeia — AI Prompt Deployment Checklist

## Registro

- [ ] Prompt ID existe.
- [ ] Versão está registrada.
- [ ] Owner está definido.
- [ ] Change Record está vinculado.

## Contrato

- [ ] Input schema validado.
- [ ] Output schema validado.
- [ ] Comportamentos proibidos definidos.
- [ ] Fallback definido.

## Avaliação

- [ ] Dataset atualizado.
- [ ] Casos comuns aprovados.
- [ ] Casos adversariais aprovados.
- [ ] Casos de segurança aprovados.
- [ ] Regressão aprovada.
- [ ] Revisão humana concluída.

## Operação

- [ ] Modelo compatível.
- [ ] Custo medido.
- [ ] Latência medida.
- [ ] Logs redigidos.
- [ ] Feature flag pronta.
- [ ] Rollout definido.
- [ ] Rollback testado.
- [ ] Alertas configurados.

## Aprovação

- [ ] Produto.
- [ ] Engenharia.
- [ ] Safety.
- [ ] Domínio, quando necessário.
- [ ] Fundador, quando alto impacto.
