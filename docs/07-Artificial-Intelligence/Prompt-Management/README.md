# Prompt Management

Esta pasta concentra a governança de prompts e mudanças de IA do Higeia.

## Uso

A norma principal define o processo.

O Prompt Specification Template deve ser copiado para cada novo prompt.

O AI Change Record deve ser preenchido em cada alteração relevante.

O Prompt Registry mantém o inventário oficial.

O checklist deve ser usado antes de staging e produção.

## Regra

Nenhum prompt de produção deve existir sem identificador, versão, schema, testes, owner e rollback.
