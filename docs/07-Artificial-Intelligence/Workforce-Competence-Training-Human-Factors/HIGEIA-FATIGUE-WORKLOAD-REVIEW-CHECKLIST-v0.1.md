# Higeia — Fatigue and Workload Review Checklist

## Workload

- [ ] Volume is measurable.
- [ ] Priority is clear.
- [ ] Critical work is limited.
- [ ] Queue aging is monitored.
- [ ] Overtime is reviewed.

## Alerts

- [ ] Alerts are actionable.
- [ ] Duplicate alerts are suppressed.
- [ ] Severity is clear.
- [ ] Escalation exists.
- [ ] False alerts are reviewed.

## Review tasks

- [ ] Review time is realistic.
- [ ] Breaks are possible.
- [ ] Rotation exists.
- [ ] Double review is used for critical cases.
- [ ] Backlog thresholds exist.

## People

- [ ] Substitute available.
- [ ] Fatigue can be reported safely.
- [ ] Stop-work authority exists.
- [ ] High-risk decisions are not made under avoidable pressure.
