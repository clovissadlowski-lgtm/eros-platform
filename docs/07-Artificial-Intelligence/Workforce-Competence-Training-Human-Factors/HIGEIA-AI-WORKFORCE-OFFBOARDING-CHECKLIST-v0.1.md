# Higeia — AI Workforce Offboarding Checklist

## Identity

- [ ] End date confirmed.
- [ ] Role transfer defined.
- [ ] Owner informed.

## Access

- [ ] Accounts disabled.
- [ ] Tokens revoked.
- [ ] API keys rotated where required.
- [ ] Sessions terminated.
- [ ] Vendor access removed.
- [ ] Repository access removed.
- [ ] Data access removed.

## Knowledge

- [ ] Handover completed.
- [ ] Documentation updated.
- [ ] Open actions reassigned.
- [ ] Incidents transferred.

## Assets and evidence

- [ ] Assets returned.
- [ ] Records preserved.
- [ ] Confidentiality obligations confirmed.
- [ ] Completion approved.
