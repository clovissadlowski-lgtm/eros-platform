## Uso

O framework define as regras gerais.

O memory schema define a estrutura mínima das memórias.

O provenance record documenta origem e transformações.

O knowledge registry controla fontes.

O dataset registry controla conjuntos de dados.

O deletion checklist garante a exclusão de derivados.

O RAG checklist controla releases de conhecimento.

## Regra

Nenhum dado deve ser usado como memória ou conhecimento sem finalidade, origem, isolamento, validade, possibilidade de correção e regra de exclusão.
