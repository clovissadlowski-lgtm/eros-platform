## Uso

O framework define as regras gerais.

O portfolio register inventaria casos.

O use case brief descreve cada proposta.

A prioritization matrix apoia prioridade.

O measurement plan define valor e métricas.

O experiment plan testa hipóteses.

O stage gate controla avanço.

O value report compara valor esperado e realizado.

O dashboard oferece visão executiva.

O periodic review registra decisões.

O sunset plan encerra casos sem valor ou inviáveis.

A WIP policy limita trabalho simultâneo.

## Regra

Nenhum caso de uso deve avançar por entusiasmo tecnológico. Problema, valor, risco, custo, owner, métricas e critério de interrupção devem estar claros.
