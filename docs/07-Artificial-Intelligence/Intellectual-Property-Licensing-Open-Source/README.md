## Uso

O framework define a política geral.

O asset register inventaria ativos.

O dependency review avalia bibliotecas.

O model terms review avalia modelos e provedores.

O dataset review avalia dados e fontes.

A allowlist policy define licenças permitidas e restritas.

A SBOM lista componentes.

O notices register controla atribuições.

A declaration controla contribuições externas.

O contractor checklist protege contratos.

O takedown record trata alegações.

O release gate protege releases.

## Regra

Nenhum ativo de terceiro deve ser incorporado sem origem, licença, versão, restrições, aprovação e evidência. Licença desconhecida deve bloquear o uso.
