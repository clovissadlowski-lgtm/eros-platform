# Higeia — Contractor and Partner IP Checklist

## Ownership

- [ ] Background IP identified.
- [ ] Foreground IP ownership defined.
- [ ] Joint IP defined.
- [ ] Deliverables defined.
- [ ] Assignment or license defined.

## Third-party assets

- [ ] Disclosure required.
- [ ] Licenses reviewed.
- [ ] Open source obligations defined.
- [ ] Subcontractors controlled.

## Confidentiality

- [ ] Confidential information defined.
- [ ] Return/deletion obligations defined.
- [ ] Publication restricted.

## Exit

- [ ] Source code delivery.
- [ ] Documentation delivery.
- [ ] Credentials transferred.
- [ ] Data returned/deleted.
- [ ] Continued-use rights defined.
