## Uso

O framework define a abordagem geral.

O obligation register inventaria requisitos.

A mapping matrix conecta requisitos a controles.

O gap record documenta lacunas.

O legal review registra interpretações competentes.

O crosswalk relaciona controles a diferentes frameworks.

O claims register controla alegações externas.

O change assessment acompanha mudanças regulatórias.

O release gate protege releases.

O readiness report comunica o estado de preparação.

## Regra

Nenhuma obrigação deve ser marcada como atendida sem controle, evidência, teste e revisão compatível. Nenhuma certificação ou conformidade deve ser declarada sem comprovação.
