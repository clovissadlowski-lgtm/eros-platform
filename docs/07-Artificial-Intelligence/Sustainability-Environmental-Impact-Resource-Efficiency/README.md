## Uso

O framework define as regras gerais.

O resource inventory inventaria consumo e custo.

A feature assessment compara valor e recursos.

A routing matrix define o modelo proporcional.

O token budget limita contexto e respostas.

O footprint review controla storage e embeddings.

O experiment record valida otimizações.

O backlog organiza melhorias.

O anomaly record trata desvios.

O vendor questionnaire avalia fornecedores.

O release gate protege novas releases.

O monthly report acompanha tendências.

## Regra

Eficiência nunca deve reduzir controles críticos de safety, privacidade, segurança ou integridade clínica. Mais computação não significa automaticamente mais valor.
