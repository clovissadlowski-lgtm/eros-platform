## Uso

O framework define a política geral.

O assessment plan organiza a avaliação.

O finding record registra problemas.

O inclusive checklist revisa linguagem, acesso e acessibilidade.

O counterfactual template compara casos equivalentes.

A matrix organiza riscos por fluxo.

O release gate controla a liberação.

O monitoring report acompanha produção.

## Regra

Nenhum fluxo crítico deve ser liberado sem avaliação de diferenças relevantes, linguagem, acessibilidade, viabilidade e possibilidade de contestação.
