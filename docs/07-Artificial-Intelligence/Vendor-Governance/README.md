## Uso

O framework define a política geral.

O inventário lista os fornecedores.

O questionário conduz a due diligence.

A avaliação registra o risco.

O plano de saída prepara substituição e exclusão.

O checklist orienta revisões periódicas.

O approval record formaliza a autorização.

## Regra

Nenhum fornecedor crítico deve receber dados reais ou acesso à produção sem inventário, avaliação, aprovação, contrato, owner, monitoramento e plano de saída.
