## Uso

O Architecture Book define a arquitetura de referência.

O Context View mostra atores e containers.

O Module Catalog delimita componentes.

O Data Flow Register mapeia dados.

O Trust Boundary Map define zonas de segurança.

O AI Flow define orchestration e revisão humana.

O Deployment Architecture orienta ambientes e releases.

Os NFRs definem requisitos de qualidade.

O ADR Register controla decisões técnicas.

## Regra

Tecnologias específicas só devem ser adotadas após decisão registrada em ADR.
