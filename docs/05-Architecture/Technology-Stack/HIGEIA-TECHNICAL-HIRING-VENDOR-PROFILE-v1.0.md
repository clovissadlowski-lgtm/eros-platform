# Higeia — Technical Hiring and Vendor Profile

## Minimum lead profile

- senior full-stack or backend engineer;
- TypeScript and Node.js;
- NestJS or comparable modular backend;
- PostgreSQL;
- AWS IAM, ECS, RDS and S3;
- authentication and authorization;
- CI/CD;
- automated testing;
- observability;
- security for sensitive data;
- ability to document architecture and decisions.

## Desirable

- health-tech;
- LGPD;
- AI API integration;
- OpenTelemetry;
- Terraform;
- threat modeling;
- incident response.

## Red flags

- proposes custom authentication;
- proposes production before staging;
- dismisses tenant isolation;
- puts secrets in `.env` committed to Git;
- calls AI provider directly from many modules;
- promises diagnosis automation in the MVP;
- proposes Kubernetes without a concrete need;
- cannot explain rollback;
- refuses to write tests or documentation.
