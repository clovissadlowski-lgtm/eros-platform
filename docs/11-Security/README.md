# 11 — Security

Esta pasta concentra documentos de segurança, privacidade e proteção de dados do Higeia.

## Documentos atuais

- `HIGEIA-SECURITY-PRIVACY-LGPD-FOUNDATION-v0.1.md`
- `HIGEIA-DATA-PROCESSING-INVENTORY-TEMPLATE-v0.1.md`

## Uso

A fundação orienta produto, engenharia e IA.

O inventário deve ser preenchido antes do Alpha fechado.

O checklist deve permanecer acessível aos responsáveis por operação e segurança.

## Regra

Nenhum usuário real deve ser admitido sem política, termos, inventário, processos de direitos, resposta a incidentes e revisão jurídica adequados.
